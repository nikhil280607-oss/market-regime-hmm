# Market Regime Detection using Hidden Markov Models

Machine Learning team project modeling latent financial market regimes
using Hidden Markov Models (HMM).

## Objective
Identify hidden market states (bull/bear, volatility regimes) from time-series data.

## Status
Theory completed. Implementation in progress (April 2026).

## Project Structure
- notebooks/ : Jupyter notebooks
- data/ : datasets
- docs/ : project documentation
